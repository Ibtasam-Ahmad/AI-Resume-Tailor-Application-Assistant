## Location Directive Generator (Use in Code)

```python
def build_location_directive(work_mode: str, job_location: str) -> str:
    loc = (job_location or "").strip()
    wm = work_mode.strip().lower()

    if wm == "remote":
        return (
            "WORK ARRANGEMENT: FULLY REMOTE.\n"
            "- REMOVE every relocation statement everywhere.\n"
            "- Emphasize remote collaboration, async communication, self-management.\n"
            "- Header tagline: replace relocation with ': Open to Remote Roles'."
        )

    if not loc:
        return (
            f"WORK ARRANGEMENT: {work_mode.upper()}.\n"
            "- Keep ONE concise availability line; do not over-emphasize relocation."
        )

    is_home = HOME_CITY.lower() in loc.lower()
    if is_home:
        return (
            f"WORK ARRANGEMENT: {work_mode.upper()} in {loc} — HOME city.\n"
            "- REMOVE all relocation language everywhere.\n"
            f"- Present as LOCAL professional in {HOME_CITY}, available immediately."
        )

    return (
        f"WORK ARRANGEMENT: {work_mode.upper()} in {loc}.\n"
        f"- Candidate lives in {HOME_CITY}, {HOME_COUNTRY} and will RELOCATE to {loc}.\n"
        f"- Replace generic relocation phrases with SPECIFIC commitment to {loc}.\n"
        f"- Update header tagline and Relocation row to name {loc}."
    )
```

## Provider-Specific API Call Templates

### OpenRouter
```python
headers = {
    "Authorization": f"Bearer {OPENROUTER_API_KEY}",
    "Content-Type": "application/json",
    "HTTP-Referer": "https://your-app.streamlit.app",
}
payload = {
    "model": "meta-llama/llama-3.3-70b-instruct",
    "messages": [{"role": "user", "content": prompt}],
    "temperature": 0.3,
    "max_tokens": 8000,
}
```

### Groq
```python
from groq import Groq
client = Groq(api_key=GROQ_API_KEY)
response = client.chat.completions.create(
    model="llama-3.3-70b-versatile",
    messages=[{"role": "user", "content": prompt}],
    temperature=0.3,
    max_tokens=8000,
)
```

## Post-Processing Steps (Same for All Providers)

1. **Strip markdown fences**: Remove ` ```latex ` and ` ``` `
2. **Extract LaTeX**: Find `\documentclass` to `\end{document}`
3. **Split changes**: Split on `===CHANGES===` marker
4. **Parse bullets**: Lines starting with `- ` after the marker
5. **Sanitize unicode**: Replace `–` with `--`, `—` with `---`, etc.
6. **Compile PDF**: Run `pdflatex` twice, or download `.tex` for Overleaf
